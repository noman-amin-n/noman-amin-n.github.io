Air Quality Analyzer

This system leverages the ESP32 microcontroller alongside the MQ135 gas sensor and the DHT11 temperature-humidity sensor to continuously monitor air quality metrics. Data is transmitted via Wi-Fi to the Blynk platform, enabling real-time remote visualization and tracking.

5-bit CPU Simulation

This project showcases the design and simulation of a basic 5-bit CPU using fundamental digital logic components. This CPU architecture was implemented with key operations, minimal instruction sets, and essential components like an ALU, RAM, Register Set, and Control Unit.
